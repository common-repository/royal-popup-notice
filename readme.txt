== Description ==

Royal PopUp Notice add a PopUp Notice & dark the website for visitor.

### Features
* "Open link in new window" option.
* Choose any google fonts.
* Choose PopUp background color or image .

== Installation ==

= Install =

1. Upload `royal-popup-notice` folder to the `/wp-content/plugins/` directory
1. In your WordPress administration, go to the Plugins page
1. Activate the Royal PopUp Notice plugin and a subpage for the plugin will appear in your WordPress Settings menu.
1. Go to the Settings > Royal PopUp Notice and configure your notice

If you find any bugs or have ideas to make it better, please let us know.
